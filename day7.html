Types Of CSS

there are 3 types of css
  1. Inline CSS
  2. Internal CSS
  3. External CSS
This is inline CSS. This is written with the tag itself via using style. See source code for more information

This is a paragraph and is selected via internal CSS. This CSS file is written in head tag using <style> tag

External CSS file is written in another file as .css file and is linked to the html using 'link' tag under head. It is useful when there are too many lines in a CSS file.
